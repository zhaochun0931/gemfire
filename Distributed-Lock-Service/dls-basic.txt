In VMware GemFire / Apache Geode, the distributed lock function is provided through the DistributedLockService (DLS).

  org.apache.geode.distributed.DistributedLockService


This service allows multiple JVMs (members) in a GemFire distributed system to coordinate access to shared resources using cluster-wide locks.
